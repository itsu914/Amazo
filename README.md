# Game → Points → Amazon Gift (MVP)

このプロジェクトは、ブラウザで遊べるシンプルなゲームをクリアするとポイントが付与され、
ポイントが閾値に到達すると自動でギフト発行（モック）を行う MVP です。

動作要件
- Node.js 18+ 推奨
- npm

セットアップ (ローカル)
1. このリポジトリをローカルに作成し、以下のファイルを保存する
2. .env を作成（.env.example をコピー）
3. 依存関係インストール:
   npm install
4. Prisma マイグレーション（SQLite を使用する場合、dev.db が作られます）:
   npx prisma migrate dev --name init
5. 開発サーバ起動:
   npm run dev
6. ブラウザで http://localhost:3000 を開く

主要フロー
- サインアップ → ログイン
- Playページでゲームをプレイ（簡易なコイン集め）
- ゲームクリア通知がサーバへ送信、サーバ側で検証・付与
- ポイントが閾値を超えると自動でギフト発行（mock provider）してコードをユーザーに表示

注意（必読）
- 本サンプルは「広告クリックを報酬化する」等の広告ネットワーク規約違反になる実装を含みません。
- 実運用ではギフト発行プロバイダ（Tremendous / Tango 等）の本番キー、TLS、KYC、追加の不正検知を必ず導入してください。
- API キー等は絶対にソース管理に平文でコミットしないでください（GitHub Secrets 等を使用）。

拡張案
- Tremendous/Tango の実装を追加（lib/giftProvider.ts に骨組みあり）
- 管理画面（赤emptionsの手動承認、ログ確認）
- 不正検知（電話認証、デバイスフィンガープリント、IPレート制限）
